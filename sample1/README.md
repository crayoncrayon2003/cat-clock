# 1. Lively Wallpaper の Install
1. Windows Store を 起動する
2. Lively Wallpaper を 検索する
3. Lively Wallpaper を インストールする

# 2. Lively Wallpaperで壁紙を起動する
1. Lively Wallpaper を 起動する
2. Lively Wallpaperのメニュバーにある「＋」を押下する
3. ポップアップダイアログで「ファイルを選択」を押下する
4. index.html を 選択する
5. OKを押下する

# 3. Lively Wallpaperで壁紙を停止する
1. Windows下部のメニューバー内、右下のタスクトレイから Lively Wallpaper のアイコンを探す
2. Lively Wallpaper のアイコンを右クリックする
3. 「壁紙を止める」を押下する

# 4. Lively Wallpaper を 止める
1. Windows下部のメニューバー内、右下のタスクトレイから Lively Wallpaper のアイコンを探す
2. Lively Wallpaper のアイコンを右クリックする
3. 「終了」を押下する
